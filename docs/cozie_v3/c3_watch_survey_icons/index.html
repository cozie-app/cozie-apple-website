Watch Survey Icons

There is two ways to add icons to the watch survey response options:

  1. Specify the filename of our built in icons without the file extension.
  2. Specify the name of an SF symbol.

SF symbols are built into iOS. You can browse them online on hotpot.ai

All built-in Symbols are listed in the table below. If you cannot find suitable icons in the table below or among SF symbols, you can post your icons in our GitHub discussion for watch survey icon requests. Please note that it might take a few days for icons to be included in the app and the new app version to be accepted in the App Store. Please note that the App Store Review Guidelines apply.
